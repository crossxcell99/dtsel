A simplistic pure javascript datetime selector


## Usage Example

```js
instance = new dtsel.DTS('input[name="date_field"]',  {
    direction: 'BOTTOM',
    dateFormat: "yyyy-mm-dd",
    showTime: false,
    timeFormat: "HH:MM:SS"
});
```
